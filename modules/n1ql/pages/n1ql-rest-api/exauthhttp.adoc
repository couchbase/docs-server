= Request with authentication - HTTP header
:description: In this example, the credentials (user="local:tweets", pass="pAss1")are given in the request header using basic authentication.
:page-topic-type: concept

{description}

.Request:
 $ curl -v http://localhost:8093/query/service \
 -d "statement=SELECT text FROM tweets LIMIT 1" \
 -u tweets:pAss1
