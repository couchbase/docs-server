== Introduction

A JavaScript function can be called like any other {sqlpp} function.
It can be executed through the Query Workbench, the REST API or the CBQ Shell

[tabs]
====
Query Workbench::
+
--
. Click on menu:Workbench[] to access the Query Editor.
+
image::javascript-udfs/workbench-menu-item.png[alt="select Workbench menu item"]

. Enter an {sqlpp} statement to call your function:
+
image::javascript-udfs/add-statement-in-editor.png[ ,600, alt="adding statement to query editor"]

. Set the `context` to `unset` so that we can run the query under the global context.
+
image::javascript-udfs/setting-context.png[alt="setting the context to unset"]

Now press the btn:[Execute] button to run the statement.

The statement will, making a call to the JavaScript function and return a result:
