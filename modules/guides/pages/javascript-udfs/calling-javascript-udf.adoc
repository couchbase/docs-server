== Introduction

A JavaScript function can be called like any other N1QL function using a suitable N1QL statement.
The Javascript function is not called directly; it is called through a N1QL reference function.

See xref:javascript-udfs/creating-javascript-library.adoc[]


== Calling the Reference function

A reference function is called through the Query Workbench, the REST API, or the CBQ shell.

====
