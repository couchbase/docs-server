= Accessing data with a browser

[abstract]
The Couchbase administrator console allows easy access to documents by using your web browser.
You can create, retrieve, update, and remove items by using this interface.

This interface should be used for informational or development purposes only (there is no REST API to access data).
In the UI, you can only edit documents that are up to 2.5 kilobits in size.

To access documents using the Web UI:

[#ol_lxc_g3m_zs]
. Navigate your browser to a cluster node.
Type the address of the cluster with the port (8091).image:webui-login.png[,650px]
+
The password is the password you used when setting up the cluster

. Once logged in, go to the Data Buckets section:image:webui-buckets.png[,650px]
. Click on the documents button of the default bucket.
(We’ll learn more about buckets later on):image:webui-documents.png[,650px]
. To retrieve a document, type in the document ID and click on "Lookup ID":image:webui-lookup.png[,650px]
+
This will take you to the document page where you can view and edit the document.
+
image::webui-editor-existing.png[,650px]

. To create a new document, go back to the ‘Documents’ section and click on ‘Create Document’image:webui-create-document.png[,650px]
. You’ll be prompted to enter a document ID:image:webui-newdoc-prompt.png[,650px]
. Once created, you can now edit your document.
Don't forget to save when doneimage:webui-editor-new.png[,650px]
