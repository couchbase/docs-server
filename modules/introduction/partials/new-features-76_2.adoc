=== Index Service

* In Couchbase Server 7.6.0, enabling file-based index rebalance prevented you from using the `WITH NODE` clause of the CREATE INDEX command.
This clause lets you choose which Index Service node contains the index.
Version 7.6.2 removes this restriction, so you can now choose which node should contain an index when your database has file-based rebalance enabled.
