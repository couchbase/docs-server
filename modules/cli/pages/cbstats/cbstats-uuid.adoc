[#cbstats-uuid]
= [.cmd]`uuid`

[abstract]
Provides the UUID for a server node.

== Syntax

Request syntax:

----
cbstats [host]:11210 uuid
----

== Description

This command provides the UUIDs for a server node.

== Options

None

== Example

*Request*

----
cbstats 10.5.2.54:11210 uuid
----

*Response*

----
uuid: 8a741dd4e271ff88c0b609dd43497d74
----
