= [.cmd]`cbq`

[abstract]
The [.cmd]`cbq` tool enables you to run N1QL queries from the command line.

== Example

The basic syntax is:

----
cbq> create primary index on `beer-sample`;
cbq> select * from `beer-sample` limit 1;
----

For a detailed explanation on logging in and running a N1QL query using [.cmd]`cbq`, see xref:n1ql:n1ql-intro/cbq.adoc#concept_s5k_qtx_1t[Running N1QL queries from a command line] .
